# Desastres Naturais: O Jogo

Desastres Naturais: O Jogo é um projeto desenvolvido na Godot Engine com foco em gerenciamento urbano e tomada de decisões durante situações de crise causadas por desastres naturais.

O jogador assume o papel de prefeito da cidade, sendo responsável por administrar recursos, responder emergências e manter a população segura diante de diferentes eventos climáticos e ambientais.

---

# Objetivo do Projeto

O projeto foi desenvolvido com fins acadêmicos e tem como objetivo simular cenários de gerenciamento de crises urbanas, incentivando planejamento estratégico e tomada de decisões.

---

# Funcionalidades

- Sistema de gerenciamento de cidade
- Eventos relacionados a desastres naturais
- Progressão de áreas e regiões
- Navegação entre cenários
- Sistema de menus
- Controle de desbloqueio de áreas
- Interação com estruturas da cidade

---

# Tecnologias Utilizadas

# Como Executar o Projeto

## Requisitos

- Godot Engine 4.x

## Passos

### 1. Clone o repositório

```bash
git clone https://github.com/Fernando-1909/Desastres-Naturais-O-Jogo.git
```

### 2. Abra o projeto na Godot Engine

- Abra a Godot
- Clique em "Import"
- Selecione a pasta do projeto
- Abra o arquivo `project.godot`

### 3. Execute o jogo

- Clique em "Run Project"

---

# Gameplay

Durante o jogo, o jogador deverá:

- Administrar recursos da cidade
- Responder a desastres naturais
- Tomar decisões estratégicas
- Liberar novas áreas
- Minimizar impactos na população
